# WriteWise AI

## Overview

WriteWise AI is an AI-powered writing assistant built by Daniyal to help users create, refine, and manage high-quality written content. Whether drafting emails, reports, social media posts, or creative writing, the application leverages modern AI models to generate context-aware responses while providing a fast, intuitive, and responsive user experience.

## Tech Stack

- Next.js 15 (App Router)
- React
- TypeScript
- Tailwind CSS
- Firebase
- OpenRouter API (Meta LLaMA)
- React Hook Form
- Zod

## Features

- AI-powered text generation using Meta's LLaMA model via OpenRouter API
- Improve previously generated responses with custom follow-up instructions
- Real-time streaming AI responses for an interactive writing experience
- Automatic history saving with options to manage or delete previous generations
- One-click copy for generated content
- Secure authentication with email, third-party providers, and guest access
- Responsive design optimized for desktop, tablet, and mobile devices
- Dark mode with automatic system theme detection
- Robust form validation for a smooth user experience
- Modular architecture focused on scalability, maintainability, and code reusability

## Local Development

### Prerequisites

Make sure you have the following installed:

- Git
- Node.js
- npm

### Clone the Repository

```bash
git clone https://github.com/Daniyalk0/writewiseAI
cd writewiseAI
```

### Install Dependencies

```bash
npm install
```

### Run Locally

```bash
npm run dev
```
